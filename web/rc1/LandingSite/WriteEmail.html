Write an email

Rollover 'Mail' in the upper gray menu bar (step 1). A drop down menu will appear. Select 'Write Message...' (step 2). A new 'Write message' tab will appear.

Use the 'Recipient' drop down menu to select a recipient (step 3). Check 'Request read receipt' if you want confirmation that your email has been viewed by the recipient (step 4). Enter a subject in the 'Subject' field (step 5). If you would like to reference a project in your email, use the 'Reference Project' drop down menu to select the desired project (step 6). If you would like to attach one of your uploaded files to an email click the 'Choose File' button (step 7). Write the content of your email in the 'Body' field (step 8). When you are finished click the 'Send Message' button (step 9).

Note: You must upload a file before you can attach it. Refer to instructions for uploading files.
