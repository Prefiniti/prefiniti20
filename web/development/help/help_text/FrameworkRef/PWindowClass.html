Overview:

Creates a new window object that can be loaded via the SessionObject.Framework.CreateWindow() function

Syntax:

windowObject = new PWindow(Handle, Title, Rect, Icon, Style, MessageHandler, Color);

Parameters:

Handle

Type: String

The string handle by which the new window will be identified.

Title

Type: String

This string will be placed in the new window's title bar.

Rect

Type: PRect() or PAutoRect()

The dimensions of the window will be determined by the dimensions of this PRect() rectangle. If a PAutoRect() is used instead, the system will determine the placement of the window automatically.

Icon

Type: PIcon()

The icon displayed in the window's application box (if applicable) and in its Desktop Manager button.

Style

Type: Numeric

Bitmask determining the desired style of the window to be created. The value of this parameter may be built by combining the desired WS_xxxxx constants together with bitwise OR.

The following style constants are available:

Style Constant Effect (when bit turned on)
WS_ALLOWCLOSE Window can be closed
WS_ALLOWMINIMIZE Window can be minimized
WS_ALLOWMAXIMIZE Window can be maximized
WS_ALLOWRESIZE Window can be resized
WS_SHOWAPPMENU Shows application menu in window title bar
WS_ENABLEPDM Allows the window to be managed by the PDM dock
WS_ROOT Window is a root window. Only the Prefiniti Window Manager can use this style.
WS_MODAL Window cannot lose focus until closed
WS_DEFAULT Shorthand for WS_ALLOWCLOSE | WS_ALLOWMINIMIZE | WS_ALLOWMAXIMIZE | WS_ALLOWRESIZE | WS_SHOWAPPMENU | WS_ENABLEPDM
WS_DIALOG Shorthand for WS_ALLOWCLOSE | WS_ALLOWMINIMIZE | WS_ENABLEPDM | WS_SHOWAPPMENU
WS_TOOLWINDOW Shorthand for WS_SHOWAPPMENU

MessageHandler

Type: Callback Function

A reference to the developer-supplied function responsible for handling system messages sent to the window. If MessageHandler is null, the framework will perform only default processing of messages on the window's behalf.

See the Framework Programmer's Guide document titled "Window Message Handling" for more information about custom message handlers.

Color

Type: PColor()

Sets the color of the window.
