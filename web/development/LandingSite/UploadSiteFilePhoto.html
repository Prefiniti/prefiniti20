Upload photo to site files

Use the site files storage space to upload company-related photos and images such as logos.

Click on 'My Files' as shown in picture (step 1). A new 'Browse Files' tab will open.

Select 'Site files' from the first drop down menu (step 2). Select 'Profile photos' from the second drop down menu (step 3). Select 'Stage Files' (step 4). Click the 'My Computer' button (step 5).

Search for and select the image you would like to upload. Click 'Start Upload' (Step 6). Close the 'Upload Status' box by clicking on the 'X' (step 7).
